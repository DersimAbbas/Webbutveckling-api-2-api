Endpoints for express in localhost.

GET /client/snuslager

makes a GET request to minimal API that will respond with all snus documents available.

Request

  • Method: GET
  • URL: http://localhost:3000/client/snuslager

Response

        [
            {
                "name": "VELO Ice Cool",
                "type": "Portion",
                "strength": "12mg/påse",
                "tobak": "false",
                "price": 49
            },
           
        ]

POST /snuslager/snus

makes a POST Request to Minimal API, that will Add a new Snus in the collection. works with postman but use the simple GUI to test directly, provided above.

Request

  • Method: POST
  • Body (required):
    •                 [
                    {
                        "name": "VELO Ice Cool",
                        "type": "Portion",
                        "strength": "12mg/påse",
                        "tobak": "false",
                        "price": 49
                    },
                   
                ]

Response

                    {
                      "message": "Snus created successfully.",
                      "data": { the data you sent with post.
                      }
                    }

Error Codes

  • 400 - Bad Request
  • 404 - Resource Not Found
  • 500 - Internal Server Error
